下さい・ください (B. 209)

Spoiler English example sentences
An auxiliary verb which indicates a polite request.
Please do something

Example sentences

(ksa).　
日本語で書いて下さい。
Please write in Japanese.
(ksb).　
英語を使わないで下さい。
Please don't use English.
(a).　
この言葉の意味を教えて下さい。
Please tell me the meaning of this word.
(b).　
夜遅く電話しないで下さい。
Please don't call me late at night.

Formation

(i) Vて 下さい  
  話して下さい Please talk
  食べて下さい Please eat
(ii) Vnegative ないで下さい  
  話さないで下さい Please don’t talk
  食べないで下さい Please don’t eat

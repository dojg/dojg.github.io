ある (2) (B. 76)

Spoiler English example sentences
Something has been done to something and the resultant state of that action remains.
Have been done; be done

Example sentences

(ks).　
それはもうジョンに話してある・あります。
It's been told to John already.
(a).　
飲み物はもう買ってあります。
Drinks have already been bought.
(b).　
窓が開けてある。
The window has been open/is open.

Formation

Vてある 話してある Has been said
 食べてある Has been eaten
